## newInstance方法

Class.newInstance() 只能够调用无参的构造函数，即默认的构造函数；

Constructor.newInstance() 可以根据传入的参数，调用任意构造函数；

new关键字是创建一个新类，只能调用无参构造；

```
A a = (A)Class.forName(“pacage.A”).newInstance();
A a = new A()；

效果是一样的
```
